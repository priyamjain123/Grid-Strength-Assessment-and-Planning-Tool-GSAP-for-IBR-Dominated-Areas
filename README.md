# Grid-Strength-Assessment-and-Planning-Tool-GSAP-for-IBR-Dominated-Areas
GSAP is a Python-based tool for automated grid strength assessment and planning in IBR-dominated networks using PSS/E. It is designed to help planners and system studies engineers assess network strength more effectively in areas with high penetration of inverter-based resources, where conventional SCR alone may not always provide a complete picture. The tool evaluates both SCR and SCR-IF for user-specified buses under base-case as well as contingency conditions, enabling a more comprehensive understanding of system behavior in weak or interaction-prone areas. It also captures the influence of fast voltage-controlling devices, including their operating modes, so that the resulting strength metrics better reflect actual network conditions. In addition to assessment, GSAP includes a planning capability that identifies optimal placement of SynCONs and GFMs (grid-forming resources) at candidate buses to help meet user-defined system strength criteria. By automating these workflows, GSAP reduces manual effort, improves study consistency, and supports faster, more informed decision-making for both operational analysis and long-term network planning.
