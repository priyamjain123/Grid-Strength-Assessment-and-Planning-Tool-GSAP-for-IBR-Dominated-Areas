# Grid-Strength-Assessment-and-Planning-Tool-GSAP-for-IBR-Dominated-Areas
GSAP is a Python-based tool for automated grid strength assessment and planning in IBR-dominated networks using PSS/E. It evaluates SCR and SCR-IF for base and contingency cases, captures the impact of fast voltage-control devices, and optimizes SynCON and GFM (grid forming resources) placement at candidate buses.
